(3) **Valid coord validation** вЂ” `hasValidCoords()` validates lat/lon are non-zero and within geographic bounds before storing. When a vehicle temporarily loses GPS, the grid continues showing the last known valid position instead of "Sem sinal".
-   **TrucksControl SWR + Fast Timeout:** API calls to TrucksControl webservice have 5s timeout (was 15s). `getCachedPositions()` implements Stale-While-Revalidate (SWR): when cache expires, stale data is served immediately while refresh runs in background. This prevents slow external API calls from blocking user requests.
-   **Performance Profiling:** Three-level profiling system: (1) `[SLOW]` вЂ” API routes >500ms logged with route/method/status/duration, stored in ring buffer (last 50). (2) `[SLOW-SUPA]` вЂ” Supabase fetch calls >500ms. (3) `[SLOW-SQL]` вЂ” Local PostgreSQL queries >200ms. Dashboard at `GET /api/health/slow-routes` (admin only) shows ranked summary with count/avg/max per route.

-   **Query Cache:** `staleTime: 30s` + `refetchOnWindowFocus: true` ensures data refreshes automatically when switching between pages/tabs.
